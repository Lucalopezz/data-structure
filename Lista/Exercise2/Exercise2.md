# Exercício 2 - Lista de Veículos

Implemente em C uma lista para armazenar a placa, marca e ano de
fabricação de veículos.
1) Defina as structs Veiculo e Lista;
2) Implemente a função void inserir (Lista *l, Veiculo v, int pos);
3) Implemente a função void remover (Lista *l, int pos).
4) Implemente a função void removerPlaca(Lista *l, char placa). <br/>
**ATENÇÃO:** a função deverá procurar em qual posição está o veículo com a
placa passada como parâmetro e, se encontrá-lo, deve removê-lo.
5) Implemente a função void mostrar (Lista *l).
