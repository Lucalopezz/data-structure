Lista duplamente encadiada
==============================
A lista duplamente encadiada é uma estrutura de dados que consiste em uma sequência de nós, onde cada nó contém um valor e dois ponteiros: um para o próximo nó e outro para o nó anterior. Isso permite que a lista seja percorrida em ambas as direções, facilitando operações como inserção e remoção de elementos.

Isso facilita a inserção e remoção de elementos em qualquer posição da lista, pois é possível acessar tanto o próximo quanto o nó anterior. No entanto, a implementação de uma lista duplamente encadiada pode ser mais complexa do que a de uma lista simplesmente encadiada, devido à necessidade de gerenciar os ponteiros adicionais.
